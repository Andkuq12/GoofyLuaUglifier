    Uglify:

    Uglififes your script with the given function name / function chain.

    POST /v1/api/uglify/<function_name>,...

    • chevron_rightHeaders
      • chevron_rightRequest Headers
        string uglifier-options?: <UglifierOptions> The options you want to used for this uglification. (optional)
        string api-key?: <string> Your API key, required to fulfill and identify your request. (optional)
      • chevron_rightResponse Headers
        string uglifier-function: <string> The used function / function chain for this uglification.
        object uglifier-options: <UglifierOptions> The options that were used for this uglification.
        number uglifier-ms-time: <number> How long the uglification took on the server.
        number x-ratelimit-limit: <number> The rate limit ceiling that is applicable for the current request.
        number x-ratelimit-remaining: <number> The number of requests left for the current rate-limit window.
        number x-ratelimit-reset: <number> The time at which the rate limit resets. (UTC epoch time in seconds)
    • chevron_rightFunction Examples

      Uglify your script with one function.

      POST /v1/api/uglify/bytestrings

      You can also specify a function chain to uglify your script with multiple functions at once. (functions are applied by order from left to right)

      POST /v1/api/uglify/bytestrings,transformnums,minify
